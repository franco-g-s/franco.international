# Projects

This section showcases various projects I'm working on or have completed, ranging from coding and research to hardware design and personal experiments.

## Coding & Software

### FDA Watch & Digest (Completed)
Automated FDA regulatory monitoring system for the biotech industry. Uses Federal Register API and eCFR API to detect regulatory changes, with AI-powered impact analysis and automated email digests.

**Tech stack:** Python, Claude AI (Anthropic), SQLite, GitHub Actions
**Key features:** 3-stage relevance filtering, CFR change detection, cross-referencing with Federal Register, automated weekly monitoring
**GitHub:** [franco-g-s/fda-watch-digest](https://github.com/franco-g-s/fda-watch-digest)

### Goschumi Invoice Automation (Completed)
Automated invoicing system for family business operations. Handles invoice generation, currency conversion tracking, and PDF creation with planned email integration.

**Tech stack:** Python, PDF generation
**Status:** Production use since September 2025

### Claude-Obsidian MCP Server Integration (Completed)
Integration project connecting Claude AI with Obsidian vault through Model Context Protocol (MCP), enabling AI-assisted note-taking and vault management.

**Tech stack:** MCP, Obsidian API, Claude AI
**Documentation:** Custom preferences and organization prompts

### QuickAdd Place Note Automation (Completed)
Obsidian automation script for creating city and country notes with automatic coordinate fetching from OpenStreetMap, fuzzy matching, and automatic icon assignment.

**Tech stack:** JavaScript, QuickAdd, Obsidian API, OpenStreetMap API
